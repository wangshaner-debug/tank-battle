# 坦克大战 - Tank Battle

一个使用 HTML5 Canvas 构建的经典坦克大战游戏，灵感来源于 FC 上的《Battle City》。

## 游戏特点

- **经典玩法**：控制坦克保卫基地，消灭所有敌军
- **多种地形**：可破坏的砖墙、坚固的钢墙
- **波次系统**：每波敌人逐渐增多，难度递增
- **敌人AI**：敌人会自动追踪并攻击玩家
- **生命系统**：3条命，基地被毁则游戏结束
- **触控支持**：移动端也可游玩

## 操作说明

| 按键 | 操作 |
|------|------|
| W/A/S/D 或 方向键 | 移动坦克 |
| 空格键 | 射击 |
| P | 暂停 |
| R | 重新开始 |

## 如何游玩

直接打开  文件即可在浏览器中游玩。
